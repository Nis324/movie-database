# 🎬 Movie Database App

A sleek, responsive React application that allows users to search for movies and watch trailers in real-time. Built as a high-performance alternative to web-based editors.

## 🚀 Features
* **Live Search:** Fast movie fetching using the OMDb API.
* **Instant Trailers:** Integrated video player with 16:9 aspect ratio.
* **Glassmorphism UI:** Modern, translucent design with smooth hover effects.
* **Responsive Design:** Optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styling:** CSS3 (Custom Properties & Flexbox)
* **Deployment:** Vercel
* **API:** OMDb (Open Movie Database)

## 📦 Local Setup
1. Clone the repo: `git clone https://github.com/Nis324/movie-database.git`
2. Install dependencies: `npm install`
3. Run the app: `npm start`
